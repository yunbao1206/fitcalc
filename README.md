# WiseClick Fit

> 10 free health and fitness calculators. Science-based tools for your wellness goals.

**Live site:** [fit.wiseclick.site](https://fit.wiseclick.site)

## Calculators

- TDEE Calculator (Mifflin-St Jeor equation)
- BMI Calculator (metric & imperial)
- Body Fat Calculator (US Navy method)
- Macro Calculator (4 diet styles)
- Calorie Deficit Calculator
- Protein Calculator
- BMR Calculator (5 activity levels)
- Water Intake Calculator
- Ideal Weight Calculator (Devine, Robinson, Miller, Hamwi formulas)
- Calories Burned Calculator (15 activities with MET values)

## Features

- Science-based formulas used by health professionals
- All calculations run in-browser — your health data stays private
- JSON-LD structured data (WebApplication + FAQPage schemas)
- Mobile responsive

## Disclaimer

These calculators provide estimates for educational purposes only. Results are not medical advice. Consult a healthcare professional before making health decisions.

## Tech

- Pure HTML/CSS/JavaScript
- Hosted on GitHub Pages
- Zero dependencies
